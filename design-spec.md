# Tripcool 迷你冷温小冰箱 - 详情页设计规范
## 极简纯净风 (Apple/Dyson 风格)

---

## 一、色彩体系

### 主色
```
白色      #FFFFFF  ████████  背景主色
浅灰      #F5F5F7  ████████  模块分隔背景
深空灰    #86868B  ████████  辅助文字
黑色      #1D1D1F  ████████  主标题文字
```

### 点缀色
```
产品黑    #2C2C2E  ████████  黑色款产品展示
产品白    #FAFAFA  ████████  白色款产品展示
产品银    #C0C5CE  ████████  银色款产品展示
```

### 使用比例
- 白色/浅灰背景：70%
- 产品主图：20%
- 文字信息：10%

## 二、字体规范

### 中文 families
```
首选：HarmonyOS Sans SC
备选：PingFang SC, Microsoft YaHei
```

### 字阶系统
```
超大标题  48px / 600  首屏主标题
大标题    32px / 600  模块标题
中标题    24px / 500  子模块标题
正文大    17px / 400  主要描述
正文      15px / 400  常规文字
辅助文字  13px / 400  参数/备注
```

### 字重使用
```
Bold (600)   标题层级
Regular (400) 正文内容
Light (300)  装饰性数字
```

---

## 三、间距系统

### 基础单位
```
4px 为最小单位
4, 8, 12, 16, 24, 32, 48, 64, 80, 120px
```

### 页面边距
```
左右边距：60px (桌面端)
模块间距：80-120px
内容间距：32-48px
```

### 留白原则
```
「宁多勿少」
每屏内容不超过 3 个信息点
图片周围至少 60px 呼吸空间
```

---

## 四、图片规范

### 产品图要求
```
- 纯白底或浅灰底
- 柔和阴影 (不强烈)
- 圆角处理 16-24px
- 高分辨率 (最小 1200px 宽)
```

### 场景图要求
```
- 明亮光线
- 浅色调环境
- 产品清晰可见
- 生活化但不杂乱
```

### 图标规范
```
- 单色线性图标
- 线宽 2px
- 尺寸 48x48px
- 圆角端点
```

## 六、设计注意事项

### ✅ 应该做的
- 保持大量留白，让内容呼吸
- 使用高质量产品图片
- 文字对齐严格遵循网格
- 色彩保持一致性
- 模块之间用背景色区分

### ❌ 避免做的
- 不要使用强烈对比色
- 不要堆砌过多信息
- 不要使用复杂渐变
- 不要添加装饰性元素
- 不要使用粗重边框
