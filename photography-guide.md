# Tripcool 迷你小冰箱 - 产品摄影参考指南
## 极简纯净风 (Apple/Dyson 风格)

---

## 一、摄影风格定位

### 参考品牌
- **Apple**: 纯白背景、柔和阴影、产品主角
- **Dyson**: 技术感、干净利落、浅灰背景
- **徕芬**: 精致质感、生活场景、暖调光线

### 核心原则
```
1. 产品清晰可见 - 占画面 40-60%
2. 背景简洁 - 纯白或浅灰渐变
3. 光线柔和 - 避免强烈对比
4. 阴影自然 - 淡阴影，不抢镜
```

## 二、必拍镜头清单

### 2.1 主图系列 (Hero Shots) - 共 14 个镜头

构图：产品 45°角，略微俯视
背景：纯白 #FFFFFF 或 浅灰渐变 #F5F5F7→#FFFFFF

## 三、摄影技术参数

### 3.1 相机设置
```
相机：全画幅 DSLR/微单 (Sony A7R 系列/Canon R5)
镜头：50mm f/1.8 或 85mm f/1.8 (产品特写)
       24-70mm f/2.8 (场景图)
光圈：f/8 - f/11 (保证景深)
ISO: 100 (最低原生 ISO)
快门：1/125s 或根据光线调整
白平衡：5200K - 5500K (日光)
格式：RAW + JPEG
```

### 3.2 灯光配置
```
主光：柔光箱 60x90cm，45°侧前方
补光：柔光箱 60x90cm 或 反光板，另一侧
轮廓光：窄条柔光箱，勾勒产品边缘
背景光：单独打亮背景 (如需要纯白)

光比：主光：补光 = 2:1 或 3:1
```

### 3.3 背景设置
```
纯白背景：白色亚克力板 或 白色无缝纸
浅灰背景：灰色无缝纸 (#F5F5F7 近似色)
渐变背景：背景纸自然过渡 或 后期处理
```

## 四、后期处理规范

### 4.1 基础调整
```
1. 校正白平衡 - 确保白色准确
2. 调整曝光 - 直方图居中偏右
3. 提升阴影 - 保留暗部细节
4. 降低高光 - 避免过曝
5. 增加清晰度 - 适度 (10-20)
```

### 4.2 产品修图
```
1. 去除灰尘/瑕疵 - 修复画笔
2. 校正变形 - 镜头校正
3. 锐化 - 智能锐化 (数量 80%, 半径 1px)
4. 降噪 - 适度 (保留细节)
5. 颜色统一 - 确保多款产品颜色一致
```

### 4.3 背景处理
```
1. 抠图 - 钢笔工具精确抠图
2. 背景替换 - 纯白 #FFFFFF 或 浅灰 #F5F5F7
3. 添加阴影 - 多层阴影叠加
   - 接触阴影：最深，紧贴产品
   - 投射阴影：渐变，向外扩散
   - 环境阴影：最浅，大范围的
```

### 4.4 输出规格
```
主图：1200x1500px @2x, @3x, JPG 85%, sRGB
详情图：1200px 宽，高度自适应，JPG 85%, sRGB
缩略图：400x400px, JPG 80%, sRGB
```

## 六、拍摄检查清单

### 拍摄前
- [ ] 产品清洁 (指纹、灰尘)
- [ ] 背景纸/板准备
- [ ] 灯光设置完成
- [ ] 相机参数设置
- [ ] 三脚架稳定

### 拍摄中
- [ ] 每个角度拍摄 3-5 张 (不同曝光)
- [ ] 检查直方图
- [ ] 放大检查对焦
- [ ] 记录灯光位置 (便于复刻)

### 拍摄后
- [ ] 备份 RAW 文件
- [ ] 初筛可用照片
- [ ] 标注用途
- [ ] 交付后期
